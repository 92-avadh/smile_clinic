# SmileCraft Dental Care 🦷

A modern premium dental clinic website built using **Next.js 15**, **TypeScript**, **Tailwind CSS**, **ShadCN UI**, and **Framer Motion**.

Designed specifically for:

* Dental clinics
* Healthcare businesses
* Client demos
* Portfolio showcase
* Agency presentations

The project focuses on:

* Modern luxury healthcare UI
* Fast performance
* Mobile responsiveness
* Smooth animations
* Conversion-focused design
* SEO optimization

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/smilecraft-dental-care.git
```

Navigate to the project folder:

```bash
cd smilecraft-dental-care
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
